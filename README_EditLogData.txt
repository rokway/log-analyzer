# Getting Started

# Prerequisites
- Ensure you have the following installed:
- Python 3.7 or higher

# Installation
- Clone this repository:

- git clone https://github.com/<your-username>/log-analyzer.git


# Navigate to the project directory:
- cd log-analyzer


# Install required dependencies (if any):
- pip install -r requirements.txt


# Usage
- Prepare your log file and place it in the project directory. 
- Ensure the file is named "Sample_Logs.txt", or update the script with your file path.

# Run the log analyzer script:
- python log_analyzer.py

View the output report in your terminal.
